# Инструменты очистки репозитория

В корне добавлен скрипт для безопасной очистки и файл с предложением изменений:

- `scripts/cleanup.ps1` — PowerShell-скрипт: сначала показывает dry-run, затем при запуске с `-Run` перемещает/удаляет файлы и сохраняет бэкап.
- `CLEANUP_PROPOSAL.md` — список предложенных изменений и объяснение.

Чтобы посмотреть что будет сделано (dry-run):

```powershell
.\scripts\cleanup.ps1
```

Чтобы выполнить очищение (выполняет резервное копирование):

```powershell
.\scripts\cleanup.ps1 -Run
```

Изменения не будут выполнены без вашего подтверждения — сначала посмотрите dry-run и подтвердите, пожалуйста.
