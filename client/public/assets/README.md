Логотипы и статические активы

- Положите сюда файлы логотипов и других публичных ассетов, которые должны быть доступны по `public` пути.
- Рекомендуемое имя логотипа: `logo.svg` — используется в `Navbar` и модальных окнах.
- Чтобы заменить логотип — просто перезапишите файл `client/public/assets/logo.svg` и перезагрузите dev-сервер.
