# GANTT Sync Rules (сохранены по запросу)

1. Когда я пишу: "Анализируем прогресс" или "Обнови Гант" → ты:
   - Открываешь файл "Диаграмма Ганта (1).xlsx"
   - Смотришь все строки с НОМЕР ИСР (43101, 43132, 43160 и т.д.)
   - Анализируешь:
        • Что уже сделано по коммитам, `tasks/archive`, отчётам
        • Текущий статус в `TASKS_DB.json` и `ROADMAP.md`
        • Реальное состояние кода (если нужно — проверить workspace)
   - Самостоятельно выставляешь РЕАЛЬНЫЙ процент выполнения в столбце "ПРОЦЕНТ ВЫПОЛНЕНИЯ" (15%, 40%, 87% и т.д.)
   - Если задача полностью готова — ставишь 1.00 (100%) и, при возможности, окрашиваешь ячейку в зелёный
   - Если просрочена — выделяешь красным (по столбцу "ДАТА ОКОНЧАНИЯ")
   - Сохраняешь файл

2. НИКОГДА не создавай новые таблицы вместо Excel — только обновляй существующий файл.

3. Файл листа: "Диаграмма Ганта"

4. Автообновление: `scripts/update_gantt.py` — пример скрипта для автоматического обновления процентов.
